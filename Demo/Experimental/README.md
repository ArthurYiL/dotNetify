### How to run

- Open `HelloWorld.sln` from Visual Studio
- Open _Project -> Set Startup Projects_ menu.
- Select _Multiple Startup Projects_ and set `AvaloniaClient`, `Server`, `WpfClient` to _Start_.
- Start (F5).
